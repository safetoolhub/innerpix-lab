
Innerpix Lab es una aplicación para analizar, previsualizar y ejecutar acciones sobre colecciones de imágenes y vídeos (principalmente orientada a fotos iOS).
Ofrece al usuario 6 herramientas principales; cada una se presenta mediante un diálogo (Dialog) específico en la interfaz para permitir la vista previa de los cambios y opciones de ejecución (por ejemplo, creación de backup antes de operaciones destructivas).

1) Limpieza de Live Photos
	- Descripción: Detecta pares foto+vídeo que forman Live Photos y ofrece opciones para limpiar, separar o eliminar el componente de vídeo asociado cuando procede. Facilita mantener sólo el elemento deseado y ahorrar espacio.
	- Dialog asociado: `ui/dialogs/live_photos_dialog.py`

2) Eliminación de duplicados HEIC/JPG
	- Descripción: Busca pares HEIC/JPG que son duplicados del mismo contenido (mismas fotos en formatos distintos) y propone eliminar o conservar según políticas (ej. conservar JPG/HEIC, por tamaño, fecha).
	- Dialog asociado: `ui/dialogs/heic_dialog.py`

3) Detección de copias exactas
	- Descripción: Encuentra archivos idénticos byte-a-byte mediante hashing. Permite revisar los pares/grupos detectados y elegir qué conservar o eliminar.
	- Dialog asociado: `ui/dialogs/exact_duplicates_dialog.py`

4) Detección de archivos similares
	- Descripción: Detecta imágenes o videos visualmente similares (mismo contenido con pequeñas variaciones: recorte, rotación, compresión). Ofrece herramientas para comparar y decidir qué mantener.
	- Dialog asociado: `ui/dialogs/similar_duplicates_dialog.py`

5) Organización de archivos
	- Descripción: Genera planes para reorganizar la colección (por fecha, por carpeta raíz, por origen como WhatsApp, etc.), mostrando la estructura propuesta antes de mover archivos.
	- Dialog asociado: `ui/dialogs/organization_dialog.py`

6) Renombrado de archivos
	- Descripción: Propone esquemas de renombrado (patrones con fecha, secuencias, metadatos) y muestra una vista previa de los nombres resultantes, detectando conflictos y proponiendo resoluciones.
	- Dialog asociado: `ui/dialogs/renaming_dialog.py`

7) Eliminación de archivos de 0 bytes
	- Descripción: Busca archivos con tamaño 0 bytes y propone su eliminación.
	- Dialog asociado: `ui/dialogs/zero_bytes_dialog.py`

Notas comunes:
- Todos los diálogos siguen el patrón de preview → plan → ejecutar y respetan la política "backup-first" para operaciones destructivas (opción de crear copia de seguridad antes de eliminar/mover/renombrar).
- La lógica de negocio está desacoplada de la UI (servicios en `services/`) y cada Dialog actúa como la interfaz de usuario para inspeccionar y aceptar los planes generados por esos servicios.
- Selección de directorio raíz: El usuario elige un directorio sobre el que se realiza un análisis recursivo que incluye ese directorio y todas sus subcarpetas; los planes y resultados mostrados en los diálogos se basan en ese conjunto de archivos escaneados.

Stack tecnológico y alcance:
 - Innerpix Lab está desarrollada en Python y es una aplicación de escritorio multiplataforma (Linux, Windows y macOS).
 - La versión de escritorio se implementa usando Qt (p. ej. PyQt6 / PySide6) para la UI.
 - Se mantiene la simplicidad y el desacoplo entre lógica de negocio y UI para facilitar una futura migración a iOS/Android. Las versiones móviles se implementarán con librerías nativas o la estrategia que se decida en ese momento.
 - Por esa razón, los servicios en `services/` son PyQt6-free y actúan como la capa reutilizable que podrá ser aprovechada por futuras implementaciones móviles.
