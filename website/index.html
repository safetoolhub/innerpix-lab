Our Privacy Commitment

🔒

SafeToolHub builds software that respects your privacy by design.

Innerpix Lab makes zero network connections. It cannot connect to the internet. There is no cloud sync, no analytics, no crash reporting, no update checking, no account system, no login — nothing leaves your machine.

Your photos and videos are deeply personal. We believe you should never have to trust a third party — including us — with access to them. That's why every single operation runs entirely on your computer, with your files, under your control.

  • No network access — The application has no HTTP client, no socket code, no API calls
  • No telemetry — No usage data, analytics, or crash reports are collected or sent
  • No cloud — Files are never uploaded, synced, or shared anywhere
  • No accounts — No login, no registration, no email required
  • Open build pipeline — Binaries are built transparently via GitHub Actions from source

What It Does

🧹 Smart Cleanup

Find and remove exact duplicates (SHA256), zero-byte files, HEIC/JPG pairs, and iPhone Live Photo remnants.

👁️ Visual Detection

Detect visually identical and similar images using perceptual hashing. Adjustable 70–95% similarity threshold.

📁 Smart Organization

Reorganize files into clean date-based folder structures. Standardize filenames with EXIF-aware date renaming.

💾 Backup First

Every destructive operation offers backup creation and dry-run simulation. Your files are always safe.

🌍 Cross-Platform

Runs natively on Linux, Windows, and macOS. Available as .deb, .rpm, Windows installer, and macOS DMG.

🔒 Zero Network

The application contains no networking code. It physically cannot connect to the internet or send data anywhere.

How It Works

  1. Select your folder — Point Innerpix Lab to any photo/video directory
  2. Automatic analysis — Multi-phase scanning indexes file types, metadata, hashes, and EXIF data
  3. Choose your tools — 8 specialized tools in 3 categories: Cleanup, Visual Detection, Organization
  4. Preview & execute — Every action shows a preview first. Back up before you delete.

FAQ

Does Innerpix Lab connect to the internet?

No. The application contains no networking code. It makes zero network connections.

Is it safe to delete files?

Yes. Every tool offers backup creation and dry-run simulation. Deleted files can go to your system trash for additional safety.

What formats are supported?

Images: JPEG, PNG, HEIC, HEIF, TIFF, BMP, WebP. Videos: MP4, MOV, AVI, MKV, WMV, FLV (metadata requires FFmpeg).

Can I undo changes?

Operations using the backup option can be restored. All deletions are logged with full paths to ~/Documents/Innerpix_Lab/logs/.
